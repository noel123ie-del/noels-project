👥 Share with Partner / Family (Cloud Sync)

Auto-sync your data to a free Firebase project so your partner sees it live on their device.

🛠️ Setup Instructions (one-time, ~10 min)
Step 1 — Create free Firebase project (5 min):
• Go to console.firebase.google.com
• Click "Add project" → name it (e.g. "thrift-sync") → continue → Create
• Disable Google Analytics (not needed)

Step 2 — Enable Firestore Database (3 min):
• Left menu → Build → Firestore Database → Create database
• Choose region (europe-west1 for Ireland)
• Pick "Start in test mode" — gives 30 days of open access (good enough for personal use)
• Click Enable

Step 3 — Get your project ID (30 sec):
• Top-left gear icon → Project settings
• Copy "Project ID" (looks like thrift-sync-12345)
• Paste it in the field above

Step 4 — Set a share code:
• Pick any phrase (e.g. "noel-anna-2026")
• Tell your partner the same code
• Tick "📤 Sync my data" on YOUR device
• Tick "👁️ View partner's data" on HER device
• Done! Your data appears on her screen within seconds.

⚠️ "Test mode" Firestore expires in 30 days — you'll need to switch to "production mode" with simple rules. Free tier handles families easily (50k reads/20k writes per day).

🚨 Emergency Kill-Switch (Admin)

Shut down all users instantly when there's a problem. Editing status.json in the repo updates every user within 5 minutes.

To shut everyone down:
1. Edit status.json (in repo or Netlify)
2. Set "active": false
3. Set "message": "Reason..."
4. Commit + push (Netlify auto-deploys ~30s)
5. All users locked within 5 min (or instantly on next reload)

Banner-only warning:
Set "banner": "Maintenance at 9pm" to show non-blocking warning.

To restore: set "active": true and "banner": "".
