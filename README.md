## ✨ Features

### Dashboard
- 📊 **Spending Overview** - Week, month, and income stats
- 📈 **6-Month Trend** - Visual spending history
- 🎯 **Category Breakdown** - Donut charts by spending type
- 📝 **Recent Transactions** - Last 20 transactions with filters

### Add Spending
- ➕ **Quick Log** - Add spending in seconds
- 🏷️ **Auto-Categorize** - Food, Transport, Shopping, Entertainment, Utilities, Other
- 📅 **Date Tracking** - Log past or today's expenses

### Insights
- 💡 **Smart Analytics** - Week/month spending summaries
- 🔄 **Recurring Detection** - Identify subscriptions (future)
- 📊 **Budget Alerts** - Set weekly budgets and get warnings

### Bank Integration (New!)
- 🏦 **Plaid Connection** - Connect AIB, Bank of Ireland, Revolut
- 🔗 **Auto-Sync** - Transactions sync automatically
- 📤 **Smart Import** - Drag & drop CSV or connect bank
- 🇮🇪 **Irish Banks** - Full support for Irish financial institutions

### Settings
- ⚙️ **Budget Configuration** - Set weekly/category budgets
- 📥 **Bank Import** - Connect banks or upload CSV
- 💾 **Export Data** - Download all your data as JSON
- 🔒 **Privacy** - 100% client-side, no servers

## 📋 Getting Started

### For Demo:
1. Visit: https://noel123ie-del.github.io/noels-project/
2. Click any of the 4 layout buttons
3. Load sample data in Settings
4. (Dark theme) Test bank connection in Settings

### To Deploy Locally:
```bash
git clone https://github.com/noel123ie-del/noels-project.git
cd noels-project
# Open index.html in your browser
```

### To Enable Live Bank Sync:
1. Sign up at plaid.com
2. Get your Client ID + Secret
3. Update dark/app.html with credentials
4. Go live with real Irish bank connections
